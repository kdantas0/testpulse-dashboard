# 📊 TestPulse Dashboard

Dashboard de acompanhamento de execução de testes e bugs por US.

## 🚀 Tecnologias
- React + Vite
- TypeScript
- Visualização com gráficos customizados

## 📊 Funcionalidades
- KPI de execução
- Cobertura de testes
- Pass rate
- Bugs por US (abertos e fechados)
- Identificação de US crítica
